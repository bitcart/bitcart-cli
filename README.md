# Bitcart CLI

To install autocomplete, download it from [here](https://raw.githubusercontent.com/bitcart/bitcart-docker/master/compose/scripts/cli-autocomplete.sh)

```bash
PROG=bitcart-cli source cli-autocomplete.sh
```

Or put it to `/etc/bash_completion.d/bitcart-cli`

For zsh autocompletion, download it from [here](https://raw.githubusercontent.com/bitcart/bitcart-docker/master/compose/scripts/cli-zsh-autocomplete.sh)

And use it:

```bash
PROG=bitcart-cli source cli-zsh-autocomplete.sh
```

Or put it to zsh site-functions directory, e.g. to `/opt/homebrew/share/zsh/site-functions/_bitcart-cli`
